# Contributing Guidelines

Thank you for your interest in contributing to Awesome AI Gadgets! This document provides guidelines and instructions for contributing.

## How to Contribute

### Adding a New Gadget

1. Make sure the gadget is AI-powered and fits one of our categories
2. Check that it's not already listed
3. Add it in the appropriate section maintaining alphabetical order
4. Follow this format:
   ```markdown
   - **[Product Name](https://product-url.com/)** - $Price - Brief description highlighting AI features
   ```

### Submission Guidelines

- **Relevance**: The gadget must have significant AI/ML capabilities as a core feature
- **Availability**: Products should be publicly available or have a clear release date
- **Information**: Include official website, price (or price range), and key AI features
- **Quality**: Links should point to official sources, not resellers or review sites

### Categories

If your gadget doesn't fit existing categories, please open an issue to discuss adding a new category before submitting.

## Pull Request Process

1. Fork the repository
2. Create a new branch (`git checkout -b add-gadget-name`)
3. Add your gadget following the format guidelines
4. Ensure your addition maintains alphabetical order within its section
5. Commit with a clear message (`git commit -m 'Add GadgetName to Consumer AI Devices'`)
6. Push to your fork (`git push origin add-gadget-name`)
7. Open a Pull Request with:
   - Clear title mentioning the gadget name
   - Description of the gadget and its AI capabilities
   - Why it's awesome!

## Updating Information

If you notice outdated information (prices, links, features), please submit a PR with updates and mention what changed in the PR description.

## Code of Conduct

### Our Pledge

We are committed to providing a friendly, safe, and welcoming environment for all contributors.

### Our Standards

- Be respectful and inclusive
- Welcome newcomers and help them learn
- Focus on what is best for the community
- Show empathy towards other community members

### Unacceptable Behavior

- Harassment, discrimination, or hate speech
- Publishing others' private information
- Spam or commercial solicitation
- Other conduct which could reasonably be considered inappropriate

## Questions?

Feel free to open an issue if you have questions or need clarification about contributing.
